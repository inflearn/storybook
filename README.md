# 인프랩 스토리북

## 준비사항

```sh
$ npm i -g yarn  # Install yarn globally
$ yarn  # Install dependencies
```

## 개발하기

```sh
$ yarn dev # Develop storybook
$ yarn deploy # Deploy storybook
```
